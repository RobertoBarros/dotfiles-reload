# AeroSpace Cheatsheet

`Caps` = `Cmd+Option+Ctrl` via Hyperkey.

## Layout

| Shortcut | Action |
| --- | --- |
| `Caps+,` | Layout accordion |
| `Caps+/` | Layout tiles |
| `Caps+=` | Resize smart +50 |
| `Caps+-` | Resize smart -50 |
| `Caps+Shift+=` | Resize smart +333 |
| `Caps+Shift+-` | Resize smart -333 |
| `Caps+R` | Flatten workspace tree |
| `Caps+Enter` | Fullscreen |
| `Caps+Shift+F` | Toggle floating / tiling |

## Window Navigation

| Shortcut | Action |
| --- | --- |
| `Caps+Down` | Focus down |
| `Caps+Left` | Focus left |
| `Caps+Right` | Focus right |
| `Caps+Up` | Focus up |
| `Caps+Shift+Down` | Move window down |
| `Caps+Shift+Left` | Move window left |
| `Caps+Shift+Right` | Move window right |
| `Caps+Shift+Up` | Move window up |

## Workspaces

| Shortcut | Action |
| --- | --- |
| `Caps+1` ... `Caps+0` | Go to workspace 1 ... 10 |
| `Caps+Shift+1` ... `Caps+Shift+0` | Move window to workspace 1 ... 10 and go there |
| `Caps+Shift+C/G/H/K/M/S/T/V/W/X/Y/Z` | Move window to the matching workspace and go there |
| `Caps+Tab` | Back and forth between workspaces |
| `Caps+Shift+Tab` | Move workspace to next monitor |
| `Caps+N` | Go to first empty workspace |
| `Caps+Shift+N` | Move window to first empty workspace |
| `Caps+[` | Go to previous non-empty workspace |
| `Caps+]` | Go to next non-empty workspace |

## Apps

| Shortcut | Action |
| --- | --- |
| `F1`, `Caps+C` | Open **Google Chrome** and go to workspace C |
| `F2`, `Caps+V` | Open **Visual Studio Code** and go to workspace V |
| `F3`, `Caps+T` | Open **cmux** and go to workspace T |
| `F4`, `Caps+H` | Open **ChatGPT** and go to workspace H |
| `Caps+F` | Open **Finder** |
| `Caps+G` | Open **Gmail** and go to workspace G |
| `Caps+K` | Open **Slack** and go to workspace K |
| `Caps+M` | Open **Google Meet** and go to workspace M |
| `Caps+S` | Open **Spotify** and go to workspace S |
| `Caps+W` | Open **WhatsApp** and go to workspace W |
| `Caps+Y` | Open **Youtube** and go to workspace Y |
| `Caps+F1` | Open localhost in **Chrome** tab |

## Service Mode

Enter service mode with `Caps+Shift+E`.

| Shortcut | Action |
| --- | --- |
| `Backspace` | Close all windows but current, then return to main mode |
| `Esc` | Reload config, then return to main mode |
| `F` | Toggle floating / tiling, then return to main mode |
| `R` | Flatten workspace tree, then return to main mode |
| `H` | Join with left, then return to main mode |
| `J` | Join with down, then return to main mode |
| `K` | Join with up, then return to main mode |
| `L` | Join with right, then return to main mode |
| `Down` | Volume down |
| `Shift+Down` | Mute volume, then return to main mode |
| `Up` | Volume up |
